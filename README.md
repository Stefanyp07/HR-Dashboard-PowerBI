# 📊 HR Dashboard Power BI

## 📌 Descripción

Dashboard ejecutivo desarrollado en **Power BI** para el análisis de indicadores estratégicos de Recursos Humanos.

Este proyecto tiene como objetivo transformar datos de RR.HH. en información útil para la toma de decisiones mediante visualizaciones dinámicas e indicadores clave de desempeño (KPIs).

---

## 🛠 Herramientas utilizadas

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📈 Indicadores del Dashboard

- 👥 Headcount
- 📊 Rotación de personal
- ⏰ Ausentismo
- 🌴 Vacaciones
- 💰 Horas Extras
- 🚻 Distribución por género
- 🎂 Edad promedio
- 📅 Antigüedad del personal
- ⛏ Personal por proyecto
- 📍 Personal por ubicación
